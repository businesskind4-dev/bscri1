## 1. Overview

This document contains the complete question bank for the Botswana Supply Chain Readiness Index (BSCRI).

### 1.1 Question Structure

Each question follows this structure:

| Element | Description |
|---------|-------------|
| **ID** | Unique identifier (Q1–Q25) |
| **Dimension** | Which of the 7 dimensions the question measures |
| **Question** | The question asked to the respondent |
| **Options** | 4 response options with scores (0–3) |
| **Rationale** | Why this question matters in the Botswana context |

### 1.2 Scoring Scale

| Score | Description |
|-------|-------------|
| 0 | No capability / Not implemented |
| 1 | Basic capability / Informal |
| 2 | Developed capability / Formalized |
| 3 | Advanced capability / Optimized |

## 2. Dimension 1: Procurement Capability

### Q1: Strategic Sourcing

**Dimension:** Procurement Capability

**Question:** Does your organization have a documented strategic sourcing process?

**Options:**
| Score | Option |
|-------|--------|
| 0 | No, we don't have any formal process |
| 1 | We have informal processes but nothing documented |
| 2 | We have documented strategic sourcing for some categories |
| 3 | We have documented strategic sourcing for all categories |

**Rationale:** Strategic sourcing is the foundation of effective procurement. Organizations without formal sourcing processes typically overpay, accept poor quality, and miss opportunities for supplier innovation. In Botswana, where many organizations rely on imported goods, strategic sourcing is critical for cost management.
