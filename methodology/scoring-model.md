## 1. Overview

The BSCRI scoring model translates raw assessment responses into meaningful scores that indicate supply chain readiness. The model is transparent, consistent, and defensible.

### 1.1 Key Principles

| Principle | Description |
|-----------|-------------|
| **Transparency** | Anyone can verify the calculation |
| **Consistency** | Same rules apply to all respondents |
| **Simplicity** | Easy to understand and explain |
| **Comparability** | Scores are comparable across organizations |

### 1.2 Score Types

BSCRI produces three types of scores:

| Score Type | Description | Range |
|------------|-------------|-------|
| **Dimension Score** | Score for each of the 7 dimensions | 0–100 |
| **Overall Score** | Weighted average of all dimensions | 0–100 |
| **Maturity Level** | Categorical interpretation of score | Foundational → World-Class |

## 2. Question Scoring

### 2.1 Individual Question Score

Each question has four options with scores:

| Score | Description |
|-------|-------------|
| 0 | No capability / Not implemented |
| 1 | Basic capability / Informal |
| 2 | Developed capability / Formalized |
| 3 | Advanced capability / Optimized |

### 2.2 Score Collection

For each assessment, the respondent selects one option per question. The selected option's score is recorded.

**Example:**

| Question | Selected Option | Score |
|----------|-----------------|-------|
| Q1 | "We have documented strategic sourcing for all categories" | 3 |
| Q2 | "We have formal evaluation criteria but don't always use them" | 2 |
| Q3 | "We use contracts but don't manage them actively" | 1 |
